# Interactive Navigation Menu

## Description
This project demonstrates an interactive navigation menu that changes its appearance when the user scrolls the page or hovers over menu items.

## Objective
To build a fixed navigation bar using HTML, CSS, and JavaScript with dynamic style changes based on user interaction.

## Features
- Fixed navigation menu visible on all pages
- Background color changes on scroll
- Hover effects on menu items
- Smooth transitions

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
- HTML structures the navigation menu
- CSS styles the menu and adds hover effects
- JavaScript detects scroll events and updates the menu style dynamically

## Output
The navigation bar remains fixed at the top and changes color when the user scrolls down the page.
